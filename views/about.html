About the Game

HotBox X is a project developed by Peter Xu for the Model Predictive Control Lab at Berkeley. The goal is to collect data to create models of human decision making. The game is very similar to Flappy Bird except that the game settings to HotBox X are variable and modifiable by the researcher. These settings include gravity, player jump height, obstacle speed, and variation in obstacle distribution, which can be based off a uniform distribution or a normal distribution with variable mean and standard deviation. There are a total of seventeen configurable settings. In every game, data is recorded whenever a new wave spawns, the player jumps, collects a reward, or collides with an obstacle.

The core game was developed in Java using the LibGDX game development library. The server was developed with the MEAN stack.
