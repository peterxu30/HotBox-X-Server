Settings Guide

The game screen is 800 x 480 pixels. (0, 0) is the top left corner of the screen and (800, 480) is the bottom right corner. Obstacles spawn from the right side of the screen and travel left.
To add new settings, just write directly into the fields. Settings are updated/created based on the game input.
All text fields must be in lowercase

Game
  • The round whose settings you are modifying.
    • Game Mode
  • penalty : Obstacle collisions result in score penalty. When score < min score, game over.
    permadeath : Obstacle collision results in game over.
    • Reward Value
  • How many points rewards are worth.
    • Penalty Value
  • The score penalty on obstacle collisions in penalty mode.
    • Min Score
  • The minimum score in penalty mode. Falling below it results in game over.
    • Player Speed
  • How much the player jumps when SPACE is pressed.
    • Player Width
  • The width of the player in pixels.
    • Player Height
  • The height of the player in pixels.
    • Player X
  • The x-coordinate of the player. The player is stationary on the x-axis.
    • Player Y
  • The y-coordinate spawn of the player.
    • Gravity
  • The value of gravity.
    • Distribution
  • The distribution type used to randomly generate game objects (obstacles and rewards) in waves.
    uniform : Generate waves based on a uniform distribution.
    normal : Generate waves based on a normal distribution.
    • Normal Mean
  • The mean of the normal distribution if the distribution is normal. Range is 0 to 9.
    • Normal SD
  • The standard deviation of the normal distribution if the distribution is normal.
    • Wave Start
  • The x-coordinate the previous wave must pass through to spawn the next wave.
    • Object Width
  • The width of obstacles
    • Object Speed
  • The speed waves travel.
    • Object Spawn X
  • The x-coordinate where waves spawn.
